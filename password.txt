Q.Write a script that asks for input word and check 1)length minimum of 8 characters. 2)Contain both small and capital alphabets. 3) contains number

CODE:
!/bin/bash

echo "Enter any word :"
read word

len=${#word}

if [[ $len -gt 7  && $word =~ [A-Za-z0-9]  ]]
     then
         echo "Correct Password"

else
         echo "Incorrect Password"
fi

OUTPUT:

Enter any word :

Correct Password
